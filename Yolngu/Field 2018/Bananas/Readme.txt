This is one of five storyboards has been created by Kilu von Prince as part of the MelaTAMP project at Humboldt-Universität zu Berlin. 

This package contains:
- 16 pdf files with the individual pictures for the bananas storyboard.
- One tex file for compiling the pictures into one story, including subtitles in Bislama and English.
- One pdf of the compiled storyboard with subtitles in English.
- One svg file containing most of the elements used to create the pictures (in Inkscape).
